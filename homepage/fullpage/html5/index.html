什么是HTML5

万维网的核心语言、标准通用标记语言下的一个应用超文本标记语言(HTML)的第五次重大修改。

简单来说，HTML5是下一代Web语言，它不再只是一种标记语言，它为下一代 Web 提供了全新的框架和平台，包括提供免插件的音视频、图像动画、本体存储以及更多酷炫而且重要的功能，并使这些应用标准化，从而使Web 能够轻松实现类似桌面的应用体验。

HTML5的优势与优点

  • 移植性强，可以在多平台上运行，降低开发成本。
  • H5应用的升级==强制升级
  • H5开发出来的网络应用（web app）可以绕过苹果的应用商店，让用户直接从网站下载。
  • 提高可用性和改进用户友好体验
  • 新增的标签助于开发定义
  • 替换FLASH和silverlight
  • 视频和音频让网站更丰富
  • 优化SEO搜索
  • 大量应用于移动应用程序和游戏

meta属性

提供有关页面的元信息（meta-information），比如针对搜索引擎和更新频度的描述和关键词

<meta name="keywords" content="HTML,ASP,PHP,SQL">

或者手机端设备

<meta name="viewport" content="width=device-width, initial-scale=1, 
 maximum-scale=1.0, user-scalable=0;">
